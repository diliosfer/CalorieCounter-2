КБЖУ-счётчик
Понять что мы делаем:
Приложение для подсчёта калорий и нутриентов (белков, жиров, углеводов) с возможностью создания пользовательской базы продуктов и сложных блюд.
Просмотр гайдиков по JavaScript и React Native

Создание концепта:
Архитектура приложения:
React Native + Expo для кроссплатформенности
Навигация через @react-navigation/bottom-tabs
Локальная база данных SQLite
Тёмный интерфейс с золотыми акцентами

Создание базы данных и основных экранов
При запуске - три основных экрана:
Экран продуктов - база простых продуктов с КБЖУ
Экран блюд - создание сложных блюд из продуктов
Калькулятор - подсчёт дневного потребления КБЖУ

Добавление функций базового минимума:

Функции управления продуктами:
Добавление продукта - название + БЖУ на 100г (калории авторасчёт)
Просмотр списка - отображение КБЖУ каждого продукта
Удаление продуктов - долгое нажатие для удаления

Функции работы с блюдами:
Создание блюда - выбор ингредиентов из базы продуктов
Указание веса каждого ингредиента
Авторасчёт КБЖУ - на 100г и общую массу
Просмотр состава - все ингредиенты блюда

Функции калькулятора:
Выбор продуктов/блюд из базы данных
Указание веса потребления
Суммирование КБЖУ за день
Редактирование веса в реальном времени

Процесс тестирования:
Проверка корректности расчётов КБЖУ

Тестирование на разных устройствах
Валидация пользовательского ввода
Оптимизация интерфейса под различные экраны

Создание алгоритмов расчётов
Формулы расчётов:
калории = (белки × 4) + (жиры × 9) + (углеводы × 4)
общий вес = сумма(вес ингредиентов)
общие калории = сумма((калории продукта / 100) × вес ингредиента)
калории_на_100г = (общие калории / общий вес) × 100

База данных SQLite

Дополнительные функции:

Подсказки интерфейса для удобства пользователей
